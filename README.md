# hostsdog

A tool to manage hosts. Down load it from [here](https://github.com/haosdent/hostsdog/releases/).

## Usage Example

### Add

```
hostsdog add company
```

### Edit

```
hostsdog edit company
```

### Switch

```
hostsdog switch company
```

### Remove

```
hostsdog rm company
```

### Fork

```
hostsdog fork home company
```

### List

```
hostsdog list
```
